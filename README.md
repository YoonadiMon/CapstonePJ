## Introduction
AfterVolt is a centralized web-based E-Waste Management System designed to address the growing global issue of electronic waste through a structured, efficient, and user-friendly digital platform.

AfterVolt aims to bridge the gap between users and formal recycling systems by providing a platform that simplifies the process of reporting, collecting, tracking, and managing e-waste. The system connects users, collectors, and administrators in a single integrated environment to improve coordination and transparency.

---

## Key Features
- 👤 User Registration & Role-Based Access (User, Collector, Admin)  
- 📦 E-Waste Pickup Request System  
- 📍 Collection Scheduling & Tracking  
- 📊 Admin Dashboard & Reporting Tools  
- 🧾 Audit Trail & Status Monitoring

---

## Technologies Used
- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL

## ⚙️ Setup Note (XAMPP Requirement)

Please ensure that **XAMPP is installed and properly configured** on your system before running this project.

Make sure the following modules are started in the XAMPP Control Panel:
- Apache
- MySQL

The project will not run correctly if these services are not active.

Additionally, ensure the database `capstonedb` (`capstonedb.sql`) has been imported via phpMyAdmin before testing the system.
